This is ShushEngine

Check out the ShushEngine GitHub repo!

This project is based on Roboteurs' SlushEngine project. You can check them out at their website or their GitHub repo.

The purpose of this project is to create a stepper motor driver that can be easily driven by a few lines of code (currently Python). This project differs from SlushEngine in that it utilizes Trinamic TMC5160 stepper drivers rather than L6480. These drivers are known to be silent due to their 256-microstepping and StealthChop&trade, hence the manipulation of the name to "ShushEngine."
